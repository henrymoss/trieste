class DiscreteSearchSpace(points: TensorType)[source]¶

Bases: trieste.space.SearchSpace

A discrete SearchSpace representing a finite set of \(D\)-dimensional points in \(\mathbb{R}^D\).

For example:

>>> points = tf.constant([[-1.0, 0.4], [-1.0, 0.6], [0.0, 0.4]])
>>> search_space = DiscreteSearchSpace(points)
>>> assert tf.constant([0.0, 0.4]) in search_space
>>> assert tf.constant([1.0, 0.5]) not in search_space
Parameters

points – The points that define the discrete space.

class Box(lower: TensorType, upper: TensorType)[source]¶

Bases: trieste.space.SearchSpace

Continuous SearchSpace representing a \(D\)-dimensional box in \(\mathbb{R}^D\). Mathematically it is equivalent to the Cartesian product of \(D\) closed bounded intervals in \(\mathbb{R}\).

Parameters

  • lower – The lower (inclusive) bounds of the box.

  • upper – The upper (inclusive) bounds of the box.

Raises

  • ValueError – If lower and upper have different shapes. Or if upper is not greater than lower across all dimensions.

  • TypeError – If lower and upper have different dtypes.
